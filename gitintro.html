GIT & GITHUB

GIT

Git is a Version Control System. Git is a Software which is maintained by Linux and it is a open-source system

By using Git We have 3 platforms

  1. GitLab
  2. Bitbucket
  3. GitHub
  • GitLab and Bitbucket are mainey used by the experts who runs Software company for Security purpose.
  • GitHub is Used by Freshers. It is easy to use for Freshers

GitHub

By using GIt and GitHub We can share code. GitHub is a Webplatform which is maintained by Microsoft and we can use it for Free or by Paid Membership.

There are several steps/stages to push the code from Local to Remote

  • Working Directory
  • Adding/Staging Area
  • Commiting Area
  • Remote Directory

Installation

  • Go to Git downloades in chrome and click install git for windows.
  • Download and run the installer.
  • Click “Next” to accept the recommended settings.
  • This will install Git and Git Bash.
  • Login to github through Email and Password
